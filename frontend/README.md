## Run the frontend

### Windows

1. Install node.js https://nodejs.org
2. Install yarn https://yarnpkg.com/

### All Systems

1. `$ yarn install`
2. `$ yarn run build`

### Run for development

`$ yarn run start`
or
`$ start.bat`

### Build for production

`$ yarn run build`
or
`$ build.bat`
